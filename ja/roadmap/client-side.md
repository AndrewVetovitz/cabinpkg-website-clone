## Poac's roadmap (client side)

### poac 0.2.0
* devDependency, testDependency
* poac build using cmake
* enhance poac internal build system
* Support release(-O3) build, -O1, -O0 -> --release | -rのみO3のエイリアス
* Support Windows (MSVC, Cygwin, Mingw)
* 内部APIの改善

### それ以降

* Global install
* ビルド済みバイナリインストール
* Support x86_64以外
