## 既存のプロジェクトに使用する

poacを既存のプロジェクトに使用する場合は、そのプロジェクトのディレクトリに移動して、`poac init`を実行します。
```
$ poac init
"./poac.yml" was created.
```

その後は、[2.1. 新しいプロジェクトの作成](creating-a-new-project.md)で行ったように、`poac install hello_world`等を実行することで、poacを使用することができます。
